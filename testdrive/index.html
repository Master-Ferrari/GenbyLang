// genby is a js-executed language built to stay friendly , simple , and fully customizable
it strips syntax down to the minimum by treating everything as a function , so instead of block-heavy structures it leans on nested calls like a ( b ( с ) )
the whole system is packaged as a single class where you define your own api , including control flow such as loops and branching , and that same class can also export an input field with highlighting and completions plus auto-generated markdown docs

1. language config
declare functions, variables, enums, and directives on a Genby instance. return the instance at the end.
